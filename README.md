# Spatial Dragon

A hand-drawn spatial drawing created with Apple Vision Pro and reconstructed from video using 3D Gaussian Splatting.

## Data

Download the Spatial Dragon data (.ply / .splat) and place it in your own scene.

- data/spatial_dragon.ply
- data/spatial_dragon.splat
